# 💰 Expense Tracker Dashboard

A modern desktop-based Expense Tracker application built using **JavaFX**, **FXML**, **Maven**, and **Object-Oriented Programming (OOP)** concepts.

This project was developed as part of academic learning and practical implementation at:

**Jaypee University of Information Technology (JUIT), Waknaghat, Himachal Pradesh, India.**

The application allows users to manage daily expenses efficiently and visualize spending patterns using interactive charts and analytics dashboards.

---

## 🚀 Features

### Expense Management
- ➕ Add new expenses
- ❌ Delete expenses
- 🔍 Search expenses by Expense ID
- 🆔 Duplicate Expense ID validation
- 💰 Automatic total expense calculation

### Data Visualization
- 🥧 Pie Chart for category-wise expense distribution
- 📊 Bar Chart for category-wise expense comparison
- 📈 Line Chart for expense trend analysis over time

### User Experience
- ✅ Confirmation dialogs before adding or deleting expenses
- ⚠️ Error handling and input validation
- 🎨 Modern JavaFX GUI with CSS styling
- 📅 Date selection using DatePicker
- 📱 Responsive dashboard layout

## ▶️ Running the Project

Clone the repository:

```bash
git clone https://github.com/lovishsharma16-blip/ExpenseTracker.git
```

Move into the project directory:

```bash
cd ExpenseTracker
```

Run the application:

```bash
./mvnw javafx:run
```

For Windows:

```bash
mvnw.cmd javafx:run
```
